How it works

Commit
Creator picks a mechanic and commits a keccak256 hash to the sa1tmyst wrapper.
02

Reveal

FAQ

Is this a different token standard?

It's a wrapper around sa1t v2. The mechanic is committed hashed on Ethereum mainnet, then revealed at graduation via the same hook flow sa1t already uses.

Can the mechanic be faked?

No. Commitment is keccak256(mechanic + salt) stored on-chain. Reveal must match the hash exactly. No admin key can change it after commit.

Does this change the curve math?

No. Curve trades, graduation, and AMM seeding are identical to sa1t v2. The wrapper only gates mechanic reveal.
